# Chemical Data, Discovery and Design

This is the course book for the first half of CHEM502, covering Lectures 1-4 and Workshops 1-4. It's a book about using Python to explore the sorts of data that you encounter when doing Chemistry, with some introductory material on Machine Learning.

You can find more course material at the CHEM502 [Canvas page](https://canvas.liverpool.ac.uk/courses/85632)
